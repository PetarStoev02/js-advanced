# JavaScript Advanced - Learn by Day

A structured, hands-on learning path for intermediate to advanced JavaScript developers. Work through challenging exercises day by day, building real skills in closures, async programming, classes, modules, and more.

## How to Use

1. Navigate to a day folder
2. Open a `.js` file and read the instructions in the comments
3. Run with Node: `node filename.js`
4. Implement the solution
5. Check your work against the `Solutions/` folder if needed
